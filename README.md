# simple-rest-api

## Description

```
Demo rest api project for codando tv 
```

## Technologies
```
Spring boot 3
Java 17
Docker
```

## Docker local instructions
```
1 - docker build -t your_tag_here/simple-rest-api .
2 - docker run -p 8080:8080 your_tag_here/rest-api
```
OR
```
docker-compose up
```
